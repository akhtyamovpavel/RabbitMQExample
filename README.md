### собрать проект
```docker-compose build```

### запустить всё
```docker-compose up -d```

### запустить конкретный сервис
```docker-compose up api```

### посмотреть логи
```docker-compose logs```

### посмотреть логи конкретного сервиса
```docker-compose logs api```


### Сервис доступен по адресу 
http://0.0.0.0:8080/

### RabbitMq доступен по адресу
http://0.0.0.0:15672/
